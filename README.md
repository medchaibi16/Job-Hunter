# Job Hunter - AI-Powered Autonomous Job Search Agent

An intelligent, self-learning job discovery system that autonomously finds AI/ML internships, learns from user preferences, and provides personalized recommendations using LLMs and vector embeddings.

## Key Features

- **Google Gemini LLM Integration** - Automated company research and intelligent email generation
- **Vector Embeddings (ChromaDB)** - Semantic job matching and similarity search
- **Preference Learning** - ML system that learns from approve/refuse decisions
- **Autonomous Agent** - Background scheduler with continuous operation
- **Smart Matching** - Multi-factor scoring (0-100) with success probability prediction
- **Web Dashboard** - Real-time Flask interface with job categorization

## Technologies

- Python, Flask, Google Gemini API, ChromaDB
- NLP, Vector Embeddings, Machine Learning
- Autonomous Agents, Background Scheduling
- Web Scraping, RESTful APIs
